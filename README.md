# Saudi Electricity Consumption Analysis

## Overview
SQL analysis of Saudi Arabia electricity consumption data from 2005 to 2024.

## Dataset
Saudi Arabia Electricity Dataset - Kaggle

## What This Project Does
1. Average Consumption by Region - Identifies which regions consume the most electricity
2. Optimal Maintenance Scheduling - Determines the best months for maintenance based on low consumption periods
3. Critical Months - Identifies months with highest electricity demand

## Tools
SQL

## Skills Demonstrated
- Data Analysis
- Subquery Usage
- Conditional Logic (CASE WHEN)
